Setup Instructions

  1. Create an Azure Communication Services resource:
    • Go to Azure Portal
    • Create a new "Communication Services" resource
    • Note down the connection string from the "Keys" section
  2. Get a phone number:
    • In your Communication Services resource, go to "Phone numbers"
    • Click "Get" and acquire a phone number with voice capabilities
    • Make sure the number supports "Make calls" and "Receive calls"
  3. Configure your environment:
    • Copy .env.example to .env
    • Add your Azure credentials to the .env file:
    AZURE_COMMUNICATION_CONNECTION_STRING=endpoint=https://...
AZURE_PHONE_NUMBER=+1234567890
CALLBACK_BASE_URL=http://localhost:5000
  4. Restart the application: 
    python app.py
Need help? Check out the Azure Communication Services documentation
